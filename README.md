# PinguStationery

PinguStationery 핑구 문구점 is a cute and interactive stationery shopping website inspired by the character Pingu, who represents the identity and mascot of the platform.

The website showcases a variety of stationery items such as notebooks, stickers, pen cases, and other accessories designed with a playful and charming aesthetic. It aims to create a smooth and enjoyable browsing experience where users can easily explore products and interact with the interface.

The project focuses on combining a friendly visual style with a simple and accessible shopping interface.

**Overview 개요**

The PinguStationery web project is a front-end focused e-commerce website that presents stationery products through a visually appealing and user-friendly interface.

The design emphasizes a soft, playful theme centered around the Pingu character while maintaining a structured layout that allows users to browse and explore products comfortably.

---

**Objectives 목표**

- Provide a visually engaging online stationery shop experience
- Create an intuitive and interactive interface for browsing products
- Demonstrate front-end web design and development skills
- Develop a cohesive design centered around a mascot-based theme

---

**Features 기능**

- Responsive layout compatible with desktop, tablet, and mobile devices
- Interactive product browsing and navigation
- Product image displays with multiple viewing angles
- Clean and playful UI inspired by the Pingu theme
- Organized product sections for different stationery items
- Smooth hover effects and visual interactions
- Simple and user-friendly shopping experience
